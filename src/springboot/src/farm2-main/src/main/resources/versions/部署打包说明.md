### 打包命令  mvn clean install -DskipTests=true


### 在tomcat下部署war包时打包
###### farm2.properties文件中的farm2.conf.web.domain.able.url 参数设置为空，等号后面空字符串‘’
###### farm2-base下pom.xml中 启用下面依赖
        <!-- 排除嵌入式Tomcat依赖项**************-->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-tomcat</artifactId>
            <scope>provided</scope>
        </dependency>
        <!-- ******************************** -->
